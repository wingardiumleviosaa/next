[Kafka] 建立 Kafka Cluster

Publish on: 2020-06-10 Classify at: DataEngineering/Kafka
Words: 236 Read:≈ 1mins
1$ cd /opt
2$ wget "http://ftp.tc.edu.tw/pub/Apache/kafka/2.4.1/kafka_2.12-2.4.1.tgz"
3$ sudo tar zxvf kafka_2.12-2.4.1.tgz
4$ sudo mv kafka_2.12-2.4.1 kafka

建立儲存 log 的資料夾，並指定權限

1$ sudo mkdir /var/lib/kafka
2$ sudo chown -R $(USER id -u):$(USER id -g) /opt/kafka
3$ sudo chown -R $(USER id -u):$(USER id -g) /var/lib/kafka

編輯server.properties設定檔

1$ sudo vi /opt/kafka/config/server.properties

修改以下參數

 1#指定 cluster 中的 broker 的唯一 id
 2broker.id=0 
 3
 4#指定監聽 ip & 阜號
 5listeners=INSIDE://[private ip]:9092,OUTSIDE://[private ip]:9094
 6advertised.listeners= INSIDE://[private ip],OUTSIDE://[public ip]:9094
 7listener.security.protocol.map= INSIDE:PLAINTEXT,OUTSIDE:PLAINTEXT
 8inter.broker.listener.name= INSIDE
 9
10#指定存放log的路徑
11log.dirs=/var/lib/kafka
12
13#指定預設的分區數
14num.partitions=3 
15
16#允許使用者刪除 topic
17delete.topic.enable = true
18
19#加入zookeeper 的位置
20zookeeper.connect=10.0.0.4:2181,10.0.0.5:2181,10.0.0.6:2181

將KAFKA設為服務

1$ sudo vi /etc/systemd/system/kafka.service

加入以下文字

 1[Unit]
 2Requires=zookeeper.service
 3After=zookeeper.service
 4
 5[Service]
 6Type=simple
 7User=root
 8ExecStart=/bin/sh -c "/opt/kafka/bin/kafka-server-start.sh /opt/kafka/config/server.properties > /opt/kafka/kafka.log 2>&1"
 9ExecStop=/opt/kafka/bin/kafka-server-stop.sh
10Restart=on-abnormal
11
12[Install]
13WantedBy=multi-user.target

啟動 kafka

1$ sudo systemctl start kafka
2$ sudo systemctl enable kafka
