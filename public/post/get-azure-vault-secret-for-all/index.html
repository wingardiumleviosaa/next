[Azure] 使用 Vault Secrets API 如何取得超過 25 筆後的資料

Publish on: 2023-02-18 Classify at: DevOps/Azure
Words: 282 Read:≈ 1mins

TL; DR

使用 Azure key vault 的 API Get 所有 secrets 時可以利用 maxresults 的參數限制回傳的筆數，預設為 25 筆，但經測試，最大的顯示筆數也就是 25 筆了，要如何取得全部的 secrets 呢? 以此篇記錄。

利用 nextLink 取得下一頁資料
