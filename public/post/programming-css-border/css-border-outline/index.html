[CSS] border & outline

Publish on: 2020-07-20 Classify at: Programming/CSS
Words: 344 Read:≈ 1mins

border 

1<div id="box1">box1</div>


1#box1 {
2    background: yellow;
3    border: 20px solid green;
4        /* 框限大小  樣式  顏色*/
5    height: 30px
6}

邊框是往外面長的，例如上面的例子 能發現到整個選取的地方高度總共是 70px

broder-style

border style 可以為外框限加上不同樣式，以下示範使用開發人員工具上面改樣式。

border-radius

可以為外框的角加上弧度。

用 border 畫圓形

可以在沒有設 border 參數下直接使用 border-radius，將值設成 50px 或是 50%，即可畫一個圓形。

用 border 畫三角形 

 1/* style.css */
 2#box1 {
 3    background: yellow;
 4    width: 100px;
 5    height: 100px;
 6    border-top: 30px solid salmon;
 7    border-bottom: 30px solid turquoise;
 8    border-right: 30px solid peachpuff;
 9    border-left: 30px solid pink;
10}

可以單獨為四個邊設定 border，每個邊會變成梯形；

當中間的元素長寬都設 0px 時，四個邊會變成三角形；

把除了要留住的三角形以外的邊以及元素本身顏色都設為 transparent 透明；

調整各邊的大小可以得道不同長邊的三角形。

Outline

Outline 跟 Border 都是邊框，不一樣的是 Outline 是外框，不會改變元件的位置，而 border 會。

Info

不過實際上比較常用 border!
