[CSS] 移入轉換文字內容

Publish on: 2020-10-16 Classify at: Programming/CSS
Words: 491 Read:≈ 1mins

移入轉換其實很簡單，不需要算什麼正負位移，只要利用 display: none 屬性就可以輕鬆達成。先來看完成的範例：

Home首頁 About關於 Services服務 Clients客戶 Contact聯絡

代碼如下：

 1<!DOCTYPE HTML>
 2<html>
 3    <head>
 4        <meta charset="utf-8"/>
 5        <style>
 6            .nav2{
 7                width: 916px;
 8                height: 20px;
 9                margin: 0px auto;
10                border-bottom: 8px red solid;
11                padding-left: 10px;
12            }
13
14            .nav2 a{
15                height: 20px;
16                width: 80px;
17                float: left;
18                margin-right: 1px;
19                text-decoration: none;
20                text-align: center;
21                background: rgb(223,223,223);
22
23            }
24            .nav2 a .en{
25                display: block;
26                color: black;
27                font-size: 10px;
28                line-height: 20px;
29            }
30
31            .nav2 a .cn{
32                display: none;
33                color: black;
34                font-size: 10px;
35                line-height: 20px;
36            }
37            .nav2 a:hover{
38                background: red;
39            }
40			<!-- 在游標移入時 .en 的樣式 -->
41            .nav2 a:hover .en{
42                display: none;
43                color: white;
44            }
45			<!-- 在游標移入時 .cn 的樣式 -->
46            .nav2 a:hover .cn{
47                display: block;
48                color:white;
49            }
50        </style>
51    </head>
52    <body>
53        <div class="nav2">
54            <!-- 利用兩個標籤轉換不同文字 -->
55            <a href="#"><span class="en">Home</span><span class="cn">首頁</span></a>
56            <a href="#"><span class="en">About</span><span class="cn">關於</a>
57            <a href="#"><span class="en">Services</span><span class="cn">服務</a>
58            <a href="#"><span class="en">Clients</span><span class="cn">客戶</a>
59            <a href="#"><span class="en">Contact</span><span class="cn">聯絡</span></a>
60        </div>
61    </body>
62<html>
